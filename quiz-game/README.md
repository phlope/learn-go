<h1>Gophercise Quiz Game</h1>

[Link to exercise guide](https://courses.calhoun.io/lessons/les_goph_01)


<h3>Run Quiz</h3>
To run first build the package from `/quiz-game` using:

```
go build
```

Then run:

```
go run main.go
```

<h3>Run Tests</h3>

Run tests for this application using:

```
go test
```
